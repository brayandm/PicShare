- [PHP 8.1](https://www.php.net/)

## Installation instructions
```bash
  git clone https://github.com/brayandm/PicShare.git
```

```bash
  cd PicShare
```

```bash
  composer install
```

```bash
  ./vendor/bin/sail up
```

```bash
  ./vendor/bin/sail artisan migrate:fresh --seed 
```
